# ColdIQ LinkedIn Writing Guide

Voice, tone, formatting, credibility tactics, and pre-publish checklist for ColdIQ content creators.

---

## Content Philosophy — The ColdIQ DNA

1. **Educate, don't sell** — teach frameworks people can implement immediately
2. **Data over opinions** — every claim backed by specific numbers
3. **Transparency wins** — share real revenue, metrics, behind-the-scenes
4. **Actionable depth** — readers should be able to DO something after reading
5. **Authority through access** — name-drop tools, leaders, insider knowledge
6. **Visuals multiply reach** — infographics get 2-3x the engagement of text-only

## Writing Voice & Tone

### DO:

- Write conversationally, in first person ("I", "We")
- Use specific numbers: "$6,000,000" not "$6M" in hooks
- Use ellipses for dramatic pauses...
- Include parenthetical asides (like this)
- Use approximations with "~" for estimates
- Mix short punchy sentences with longer explanations
- Use fragments for emphasis. Like this.
- One idea per line for bullet points
- Use ">" for greater than: "> 50,000"
- Use "+" for approximate: "100+"
- Use commas in large numbers: "$6,000,000"

### DON'T:

- Use corporate jargon ("leverage synergies", "thought leadership")
- Make vague claims without numbers
- Write in third person
- Use flowery or hype language ("amazing", "revolutionary", "game-changing")
- Oversell — let numbers do the talking
- Add emojis to every line (use sparingly: 1️⃣ for sections, 👇 for CTAs)
- Start with context — always start with the result

### Sentence Rhythm:

```
Short punchy line.
Another one.

Then a longer sentence that adds context and explanation for people who want more depth.

Then back to short. Like this.
```

## Formatting Rules

### Visual Hierarchy:

1. **Emoji section headers**: 1️⃣ 2️⃣ 3️⃣ for major sections
2. **Arrows**: → for actions/processes
3. **Dashes**: - for tool lists/items
4. **Spacing**: Double line break between sections, single within

### Bullet Formats:

```
Main point:
- Sub-point (context or detail)
- Sub-point (context or detail)

For [outcome], use:
→ Action item
→ Action item
```

### Character Limits by Post Type:

| Post Type | Character Range |
|-----------|----------------|
| Educational posts | 2,000-2,800 chars |
| Story/personal posts | 1,500-2,200 chars |
| Comment-gated posts | 1,200-1,800 chars |
| Carousel companion text | 800-1,500 chars |

## Credibility Building Tactics

### 1. Name Dropping (with context)

```
"Mich Lieben just crossed 65,000 followers. His stack:"
"As [Name] at ColdIQ, I run outbound for 80+ B2B clients."
```

### 2. Social Proof

```
"We publish ~200 LinkedIn pieces per month"
"Our team combines 300,000+ followers"
"This generated 2.5M+ monthly social impressions"
```

### 3. Results-First

```
"LinkedIn generates 60% of all their meetings"
"This campaign added $2M in pipeline in 90 days"
```

### 4. Transparency

```
"Exact revenue: $7M ARR"
"Tool costs: $1,500-2,000/mo total stack"
"Team: 35 FTEs + 12 fractional"
```

## Engagement & Closing Patterns

### P.S. Questions (most effective closers):

```
P.S: What tool should I add to this stack?
P.S: How are your cold email campaigns doing right now?
P.S: What's the ONE tool you can't live without?
P.S: Which part of this system would you want more detail on?
```

### Comment Gating:

```
Comment "INTENT" below and I'll share it.
Comment "STACK" and I'll send you the full breakdown.
Want the template? Drop a 🔥 below.
```

### Continuation Tease:

```
I'm running out of space... Continuing in the comments 👇
Make sure to follow so you don't miss the next breakdown.
```

## When to Mention ColdIQ

- **Subtle:** "At ColdIQ, we use..." or "Including ourselves, at ColdIQ!"
- **Direct:** Only in hiring posts, company results, or when naturally relevant
- **Never:** In the hook, as the main focus, or without adding value first

## Pre-Publish Checklist

- [ ] Hook includes specific number or bold claim
- [ ] Opens with result, not context
- [ ] Every major claim backed by specific data
- [ ] Includes 3+ tool mentions or name drops
- [ ] Character count in the sweet spot for this post type
- [ ] Emoji section headers (1️⃣ 2️⃣ 3️⃣) if applicable
- [ ] Mix of dashes (-) and arrows (→) for visual variety
- [ ] Ends with "P.S:" question or engagement prompt
- [ ] Written in first person ("I", "We")
- [ ] Scannable — not walls of text
- [ ] If visual: design brief is complete and handed off
- [ ] No selling in the hook
- [ ] Would I stop scrolling for this?
