# LinkedIn Post Structure Templates

5 proven post templates with character counts, layouts, and use cases. Based on analysis of ColdIQ posts achieving 200-500+ reactions.

---

## Template A: The Comprehensive Breakdown (2,500-2,900 chars)

**Best for:** Tech stack reveals, company deep dives, system architectures

```
[HOOK with specific numbers]

[1-2 line credibility builder]

[Brief preview of what's coming]

1️⃣ [SECTION NAME]

[Brief intro]

We use:
- Tool A (specific use case)
- Tool B (specific use case)
- Tool C (specific use case)

2️⃣ [SECTION NAME]

[Context or insight]

For [specific outcome]:
→ Action with detail
→ Action with detail
→ Action with detail

[Continue for 5-7 sections]

P.S: [Engagement question]
```

## Template B: The Listicle Framework (2,000-2,500 chars)

**Best for:** Lessons learned, principles, frameworks, hiring advice

```
[HOOK with journey/result]

Here are X things I learned:

1. [Principle Name]

[2-3 sentence explanation]

→ Actionable sub-point
→ Actionable sub-point
→ Actionable sub-point

2. [Principle Name]

[Pattern explanation]

One way I do this:
→ Action step
→ Action step

[Continue for 5-8 principles]

P.S: [Question about their situation]
```

## Template C: The Case Study / Before-After (2,000-2,500 chars)

**Best for:** Client results (anonymized), campaign breakdowns, ROI stories

```
[HOOK: Bold result + metric]

[Context: What the situation was BEFORE]

Here's what was broken:
→ Problem 1 (with number)
→ Problem 2 (with number)
→ Problem 3 (with number)

Here's what we changed:
→ Fix 1 (specific action + tool)
→ Fix 2 (specific action + tool)
→ Fix 3 (specific action + tool)

The results after [timeframe]:
→ Metric 1 (before → after)
→ Metric 2 (before → after)
→ Metric 3 (before → after)

TAKEAWAY:

[1-2 sentence principle]

[Follow-up question or CTA]
```

---

## Template D: The Comment-Gated Lead Magnet (1,500-2,000 chars)

**Best for:** Maximum engagement, list building, lead magnet distribution

```
[HOOK: Bold claim with specific number]

[2-3 lines explaining WHY this matters]

[Brief preview of what you'll get — 3-5 bullet teaser]

→ Teaser point 1
→ Teaser point 2
→ Teaser point 3

[The "But here's the BONUS" escalation]

[Final CTA: Comment [KEYWORD] and I'll share it]
```

**Why it works:** Comment-gating drives 3-5x normal comment volume. The algorithm sees massive comment count → boosts distribution. Best-performing posts on ColdIQ accounts consistently use this template.

---

## Template E: The Conversational Story (1,800-2,500 chars)

**Best for:** Leadership, agency scaling, relationship content, personal takes

```
[HOOK: Punchy statement or provocative claim]

[Short personal story or observation — conversational, no fluff]

Here's the thing:

[Core insight in 2-3 sentences]

[Supporting points — short, punchy]
- Point 1
- Point 2
- Point 3

[Reflection or bigger-picture takeaway]

[Close with personal conviction, not a CTA]
```

## Template Selection Guide

| Post Type | Template | Optimal Length |
|-----------|----------|---------------|
| Tech stack reveals | A | 2,500-2,900 chars |
| System architectures | A | 2,500-2,900 chars |
| Lessons learned | B | 2,000-2,500 chars |
| Frameworks / principles | B | 2,000-2,500 chars |
| Client results / ROI | C | 2,000-2,500 chars |
| Campaign breakdowns | C | 2,000-2,500 chars |
| Resource giveaways | D | 1,500-2,000 chars |
| Lead magnet distribution | D | 1,500-2,000 chars |
| Personal stories | E | 1,800-2,500 chars |
| Leadership / philosophy | E | 1,800-2,500 chars |
| Carousel companion text | Any (shortened) | 800-1,500 chars |
