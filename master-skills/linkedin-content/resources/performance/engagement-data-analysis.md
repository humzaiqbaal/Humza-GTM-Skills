# LinkedIn Engagement Data Analysis

Performance insights from 86+ posts analyzed across ColdIQ's founder accounts (Mich Lieben, Kenny Damian) and COO (Alex Vacca).

---

## Mich Lieben — 43-Post Analysis (65K+ followers)

### Key Metrics

- **Average:** 326 reactions, 174 comments per post
- **Best:** 789 reactions, 424 comments (personal vulnerability story)
- **Average character count:** 2,192
- **Top performer chars:** 2,000-2,300 (story/vulnerability)

### Engagement Tier Breakdown

| Tier | Reactions | What Triggers It |
|------|-----------|-----------------|
| S-Tier (700+) | 789 | Personal vulnerability, failure stories, founder journey |
| A-Tier (500+) | 520-574 | European tech angle, annual results with specific $ |
| B-Tier (350-500) | 370-518 | Revenue hooks, comment-gated resources, team involvement |
| C-Tier (250-350) | 253-348 | Tech stack breakdowns, tool comparisons, how-to's |
| D-Tier (<250) | 161-244 | Product launches, category overviews, short promos |

### Hook Patterns Ranked by Engagement

1. **VULNERABILITY** — Personal failure/journey stories (789 avg reactions)
2. **COMMENT-GATED + AI** — "Use this Claude setup, it handles 90%" + resource giveaway (692 comments — highest comment count)
3. **REVENUE + SPECIFIC $** — "$4M ARR", "$534,000 added in January" (500+ reactions consistently)
4. **EUROPEAN ANGLE** — "But have you seen what's being built in Europe?" (520-545 reactions)
5. **TEAM/CULTURE** — "We get the whole team involved" (383 reactions, but SHORT post — 851 chars with carousel)
6. **TECH STACK REVEALS** — "Here's every tool we use" (375 reactions, reliable performer)
7. **TIME SAVINGS** — "What took hours now takes minutes" (366 reactions)

### #1 Insight: Vulnerability Outperforms Everything

The highest-performing post (789 reactions, 424 comments) was a personal story about 7 failed businesses — clothing brand, lottery, newsletter failures — and what each taught him. NOT a tech stack post.

### Character Count Sweet Spots

| Post Type | Optimal Length | Engagement |
|-----------|---------------|------------|
| Carousel companion text | 800-900 chars | 370-383 reactions (high for length) |
| Story/vulnerability | 2,000-2,300 chars | 789 reactions (highest) |
| Tech stack breakdown | 2,400-2,900 chars | 300-575 reactions |
| Educational how-to | 2,500-2,950 chars | 250-350 reactions |
| Comment-gated resource | 2,300-2,400 chars | 439 reactions + 692 comments |

### Top-Performing Patterns

**Pattern 1: Full Company Breakdown**
- Hook: "$7M ARR. Here's every tool we use."
- Structure: Department-by-department with specific tools + team members
- Engagement: 300-375 reactions, 150-165 comments

**Pattern 2: Educational How-To**
- Hook: "How the top 1% make cold email work in 2026"
- Structure: 8-9 numbered principles with sub-bullets
- Engagement: 250-280 reactions, 140-145 comments

**Pattern 3: AI/Tool Deep Dive**
- Hook: "I spent 300+ hours using Claude Code for GTM"
- Structure: Numbered tool list with specific use cases + API integrations
- Engagement: 280-310 reactions, 170+ comments

**Pattern 4: European Tech Angle**
- Hook: "Silicon Valley gets all the headlines. But Europe..."
- Structure: Company-by-company breakdown with use cases
- Engagement: 500+ reactions (highest for non-vulnerability)

**Pattern 5: LinkedIn Growth Meta-Content**
- Hook: "I just crossed 65,000 followers. Here's every tool I used."
- Structure: Category-by-category tool stack
- Engagement: 300+ reactions

## The Engagement Multiplier Formula

Posts that combine these 5 elements get the highest engagement:

1. **Specific $ in the hook** — not "grew revenue" but "$534,000 added in January"
2. **Contrarian or surprising angle** — Europe > Silicon Valley, failures > wins
3. **Visual companion** — infographic or carousel doubles reach
4. **Comment gate** — "Comment X to get it" drives 3-5x normal comment volume
5. **Name drops with follower counts** — builds network effect
